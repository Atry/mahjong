项目简介

Max-Prob-Segment是最大概率分词的Java实现，不仅是分词，还包括：

  • 词性标注
  • 拼音标注
  • 词义标注

同时这是一个Open Source项目，代码、词典和概念网络都由大众维护，期待您参与贡献。

中文分词Web Api

Max-Prob-Segment含有一个Web Api:
http://max-prob-segment.herokuapp.com/api/segment

其支持GET和POST方法访问，使用时请添加Header content-type:"application/json;charset=utf-8"和accept:"application/json"

GET /api/segment?sentence={分词测试}

POST /api/segment?params...\n\n
{将文本放在请求的Body中}
